# JosiePaper

This is the fork of PaperMC that UltraVanilla uses.

## Build

Linux / macOS:

```bash
./gradlew applyAllPatches && ./gradlew createReobfBundlerJar
```

Windows:

```powershell
.\gradlew.bat applyAllPatches
.\gradlew.bat createReobfBundlerJar
```
