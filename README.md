# JosiePaper

This is the fork of PaperMC that UltraVanilla uses.

## Build

Linux / macOS:

```bash
./gradlew applyPatches && ./gradlew createReobfBundlerJar
```

Windows:

```powershell
.\gradlew.bat applyPatches
.\gradlew.bat createReobfBundlerJar
```
